Export of Output

Overview

Once the detection and post-processing steps are complete, Civic Builder enables you to export your results in industry-standard formats like .OBJ and .FBX. These formats ensure compatibility with most 3D modeling, simulation, and rendering pipelines.

Export Options

  • .OBJ: A widely-used format for 3D geometry, suitable for applications like 3D printing, simulations, and rendering workflows.
  • .FBX: Ideal for more complex pipelines that may involve animations, materials, or integration with game engines like Unity or Unreal Engine.

How to Export

  1. Ensure all detection and post-processing steps have been completed.
  2. In the Civic Builder GUI, select the desired export format (.OBJ or .FBX) using the provided dropdown menu or settings panel.
  3. Specify the output directory where the exported files will be saved. This should be a directory accessible on your system.
  4. Click on the Export button to finalize the process.

Once the export is complete, you can navigate to the specified output directory to retrieve your files.

Integration with Blender 4.0

Civic Builder comes with a dedicated Blender 4.0 Plugin that integrates seamlessly with Blender workflows. This plugin simplifies the process of importing exported .OBJ and .FBX files into Blender, allowing for quick adjustments, visualization, and rendering.

For more details, refer to the next section on Using the Blender Plugin.

Tips for Smooth Export

  • Double-check that the correct output directories are selected before starting the export.
  • If working with high-resolution or large datasets, ensure your system has sufficient storage and memory resources.
  • Choose the format that aligns best with your project pipeline to avoid additional conversion steps.

With Civic Builder's robust export functionality, you can effortlessly integrate accurate 3D footprints into your preferred modeling or simulation tools.
