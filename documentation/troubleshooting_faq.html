Troubleshooting & FAQs

Troubleshooting

Encountering issues while using Civic Builder? Here are some common troubleshooting steps:

1. General Troubleshooting

  • Ensure you are using a compatible web browser or operating system version.
  • Check your internet connection for stability, especially when using the web application.
  • Clear your browser's cache and cookies to resolve unexpected behavior.
  • Restart the application or browser if you encounter performance issues.

2. Web Application Issues

  • If the web application is not loading correctly, verify your login credentials and try again.
  • Ensure JavaScript is enabled in your browser settings for full functionality.
  • Check the server status or maintenance updates from the Civic Builder website.

3. Standalone Component

  • If using the standalone version, ensure your operating system meets the minimum requirements.
  • Verify the installation steps and dependencies as per the installation guide.
  • Consult the installation log files for any specific error messages.

Frequently Asked Questions (FAQs)

Below are some frequently asked questions about Civic Builder:

1. Standalone startup is too slow!

The standalone application is still in development and will be optimized on release. The web application version will feature much faster load times.

2. Feature detection is not accurate enough!

Civic Builder offers a multitude of different pre-trained models for your project needs. Feel free to experiment, or submit a feature request.

3. How can I contribute?

Feel free to submit pull requests for any issues on the github page, or support through Patreon!

If you have a question that isn't answered here, please feel free to contact us for support.
