Using the Web Application

Overview

The Civic Builder web application provides a convenient platform for creating detailed 3D cityscapes without the need for local software installation. Hosted on a secure server and powered by Python Flask, it offers both free and paid user options.

User Authentication

Before using Civic Builder's web application, users must authenticate themselves:

  • Registration: Navigate to the Civic Builder website and sign up for a free account. No credit card information is required for registration.
  • Login: After registration, log in securely using your username and password.
  • Account Types: Choose between a free account with basic features or upgrade to a paid account for access to premium tools and enhanced functionalities.

Getting Started

Once logged in, follow these steps to start using Civic Builder:

  1. Dashboard: Upon login, you will be directed to your personalized dashboard.
  2. Project Creation: Start a new project by clicking on the "Create New Project" button.
  3. Interface: Familiarize yourself with the intuitive user interface designed for ease of navigation and efficient workflow.
  4. Tools and Features: Explore various tools such as importing imagery, model selection, and real-time rendering options.
  5. Saving and Exporting: Save your work and export projects in various formats suitable for further analysis or sharing.

Features

The web application includes essential features for urban environment creation:

  • Real-time Rendering: Visualize cityscapes in real-time with adjustable parameters.
  • Data Import: Easily import GIS data and imagery to enhance modeling accuracy.
  • Export Options: Export finished projects in formats compatible with popular 3D modeling software.
  • Collaboration Tools: Share projects with collaborators and manage permissions seamlessly.

With Civic Builder's web application, users can leverage powerful cityscape modeling capabilities directly from their web browser, making it an ideal solution for professionals and enthusiasts alike.
