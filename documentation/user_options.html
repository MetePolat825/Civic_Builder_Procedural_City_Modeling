User Options in Civic Builder Blender Plugin

This page provides a detailed overview of all the user options available in the Civic Builder Blender plugin. Each option is explained with its purpose, default values, and examples.

General Options

Building Height

Adjust the height for extruded buildings. This value is used when applying the Extrude Building operation.

  • Property: building_height
  • Default Value: 25.0

Example usage:

Building Height: 30.0

Texture Paths

Specify file paths for roof, wall, and imagery textures. These textures are applied to imported objects.

  • Properties:
    • roof_texture_path: Path to the roof texture file
    • wall_texture_path: Path to the wall texture file
    • imagery_texture_path: Path to the imagery texture file
  • Default Value: None (must be set by the user)

Example usage:

Roof Texture Path: C:\Textures\roof.jpg
Wall Texture Path: C:\Textures\wall.jpg
Imagery Texture Path: C:\Textures\imagery.jpg

UV Scale

Scale the UV mapping for roofs and walls during texture application. Adjust to fit texture resolution.

  • Properties:
    • roof_uv_scale: Scaling factor for roof UV mapping
    • wall_uv_scale: Scaling factor for wall UV mapping
  • Default Values:
    • roof_uv_scale: 40.0
    • wall_uv_scale: 40.0

Example usage:

Roof UV Scale: 50.0
Wall UV Scale: 30.0

Key Features

Extrude Building

Extrudes selected building footprints to the height defined in building_height.

  • Operator: object.extrude_building

Apply Textures

Applies roof, wall, and imagery textures to the selected object. Materials are assigned based on face orientation.

  • Operator: object.apply_textures

Regenerate UVs

Regenerates UV mappings using Cube Projection for roofs, walls, or imagery.

  • Operators:
    • object.regenerate_roof_uvs
    • object.regenerate_wall_uvs
    • object.regenerate_imagery_uvs

Import Feature Data

Import building footprints or imagery in .obj format. Automatically aligns geometry.

  • Operators:
    • object.import_footprint_obj
    • object.import_imagery_obj

Help and Documentation

Opens local documentation for the Civic Builder plugin.

  • Operator: object.open_documentation

Notes

Tip: Save frequently while using the plugin, and keep your file paths organized for better project management.
