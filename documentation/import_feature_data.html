Importing Feature Data

The custom add-on provides tools to import feature data like building footprints and imagery files in .obj format. This guide explains how to use these tools to bring feature data into Blender.

Step-by-Step Guide

1. Access the Import Tools

Navigate to the Civic Builder tab located in the N-Panel of Blender's 3D Viewport. You will find options for:

  • Importing footprint geometry
  • Importing imagery geometry

2. Select a File

Click the relevant import button to open Blender's file browser. Locate the .obj file you wish to import, and ensure it matches the supported format.

3. Import the Data

After selecting the file:

  • Footprint Geometry: The geometry will be imported as a mesh and rotated 90 degrees on the X-axis for alignment.
  • Imagery Geometry: The same alignment process ensures the imported imagery fits the Blender coordinate system.

A success message will confirm the import. If there's an error, ensure the file path and format are correct.

4. Applications of Imported Data

Use imported feature data to:

  • Generate procedural buildings by extruding footprints
  • Create terrain maps or background imagery

Notes

Tip: Maintain a consistent file structure and use clear naming conventions for easier management of your feature data files.
