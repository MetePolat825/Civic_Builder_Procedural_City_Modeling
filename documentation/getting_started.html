Getting Started with Civic Builder

What You Need to Get Going

To begin using Civic Builder effectively, ensure you have the necessary tools and resources:

Hardware Requirements

Civic Builder is designed to run on standard desktop or laptop computers. While specific hardware requirements can vary based on the complexity of your projects, we recommend:

  • Processor: Multi-core processor (Intel i5 or equivalent recommended)
  • RAM: 8GB or more
  • Graphics: Dedicated GPU recommended for optimal performance

Software Requirements

Depending on whether you choose the web application or standalone offline version, software requirements differ:

  • Web Application: Accessible through any modern web browser with JavaScript enabled. No additional software installation is required.
  • Standalone Offline Version: Available for Windows operating systems only. Ensure your system meets the minimum requirements specified in the installation instructions.

Data Requirements

To create accurate 3D models, you'll need:

  • Imagery: High-resolution satellite imagery or aerial photographs of the area you intend to model.
  • GIS Data: Optional but recommended for precise geospatial alignment and feature extraction.

Installing Civic Builder

Web Application

Using Civic Builder as a web application is straightforward:

  1. Visit the Civic Builder website at [to be added!].
  2. Create an account or log in to your existing account. A free account is available with a limited trial version of the software.
  3. Begin using the application directly from your browser.

Standalone Offline Version

For users preferring an offline experience on Windows:

  1. Download the Civic Builder installation package from the official website.
  2. Run the installer and follow the on-screen instructions.
  3. Launch Civic Builder from your desktop after installation.

Using Civic Builder

Web Application

The web application provides a user-friendly interface for creating 3D models:

  1. Import your satellite imagery or aerial photos directly into the application.
  2. Select and configure 3D model parameters such as building types, roads, and terrain features.
  3. Generate and preview your 3D cityscape in real-time.
  4. Export the finalized model for further use or integration.

Standalone Offline Version

The standalone version offers more flexibility and control:

  1. Import local imagery or connect to GIS databases for data integration.
  2. Utilize advanced features for detailed feature extraction and customization.
  3. Process and refine your 3D models with tools optimized for offline performance.
  4. Export models in various formats compatible with industry-standard software like Blender.

Whether you're new to 3D modeling or an experienced urban planner, Civic Builder empowers you to create realistic cityscapes efficiently. Explore our comprehensive documentation and community resources to maximize your experience.
