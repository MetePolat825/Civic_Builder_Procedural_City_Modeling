Blender Plugin Overview

Introduction

The Civic Builder Blender plugin streamlines the integration of 3D models generated in Civic Builder into Blender. The plugin provides tools for importing geometry, applying textures, and generating 3D buildings with ease.

Installing the Plugin

  1. Download the Blender Plugin from the Civic Builder website.
  2. Open Blender, navigate to Edit → Preferences → Add-ons.
  3. Click Install..., then locate and select the plugin file (.py).
  4. Enable the plugin by checking the box next to "Civic Builder Plugin".

Installing the Civic Builder Plugin.

Importing Geometry

The plugin supports importing .OBJ files for footprints and imagery:

  1. Open the Civic Builder tab in the Blender Sidebar (N key).
  2. Click Import Footprint Geometry or Import Imagery Geometry, then select the desired .OBJ file.
  3. The geometry will appear in the Blender viewport.

Importing geometry using the plugin.

Extruding Buildings

  1. Select a footprint object in the viewport.
  2. Specify the desired building height in the "Building Height" field under the Geometry Generation section.
  3. Click Extrude Building to generate the 3D structure.

Extruding a 3D building from a footprint.

Applying Textures

  1. Set the file paths for roof, wall, and imagery textures under the "Set Texture Paths" section.
  2. Select the target object, then click Apply Textures to map the textures onto the object.

Applying textures using the plugin.

Regenerating UV Mapping

The plugin offers tools to regenerate UV mappings for better texture alignment:

  1. Select the object whose UVs need regeneration.
  2. Choose the appropriate regeneration tool (Roof UVs, Wall UVs, or Imagery UVs).
  3. Click the corresponding button to apply cube projection mapping.

Regenerating UV mappings for better texture alignment.

Help and Documentation

Access documentation directly from the plugin interface by clicking Open Documentation under the "Help and Documentation" section. This opens the local documentation file in your default browser.

Opening the Civic Builder documentation from Blender.
