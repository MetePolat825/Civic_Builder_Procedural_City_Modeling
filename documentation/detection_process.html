Detection Process

Step-by-Step Detection Process

After setting paths for input and output files, we are now ready to extract features. The process in this window follow a simple structure:

  • • Press "Run Feature Extraction" to extract features.
  • • Monitor the Extraction Log and the progress bar. These will let you know once extraction is completed.
  • • Once extraction is completed, either test the output in Blender automatically, open output directory for manual usage in other programs, or further configure Civic Builder and re-run the application as many times as needed.

Step 1: Run Extraction Window

Start by selecting the "Run Extraction" window to begin the feature extraction process.

Step 2: Monitor Progress

Observe the progress in the log on the right side of the screen. It provides real-time updates on any errors, progress percentage, and estimated remaining time.

Step 3: Finished Extraction

Once extraction completes, the application will provide feedback and suggest the next steps.

Step 4: Completion and Output

Once the extraction is complete, the log will indicate the status. Use the "Open Output Directory" button to navigate directly to the generated output.

Next Steps:

With these steps done, extraction has been completed. You can re-run the application as many times as needed after this point, and fine tune the output using the configuration windows.
