Acknowledgments

Civic Builder extends its gratitude to the following individuals, organizations, and projects for their valuable contributions:

Detectron 2 Model

The detection model used in Civic Builder is based on Detectron 2, an open-source deep learning framework developed by Facebook AI Research (FAIR). This model enables accurate object detection and feature extraction in urban environments. More information can be found here.

GitHub Repository - Buildings Detection with GeneralizedRCNN

We would like to recognize the contributions from the repository maintained by Asv53, specifically the Buildings Detection with GeneralizedRCNN project. This repository provided foundational code and models that were adapted and trained for early versions of Civic Builder.

Individual Datasets

Roboflow Datasets

The Civic Builder project utilizes datasets provided by Roboflow, which have been instrumental in training our models for accurate building detection and urban environment modeling.

OpenStreetMap Datasets

We acknowledge OpenStreetMap (OSM) for their comprehensive geospatial data, which forms the foundational data source for the creation of realistic cityscapes in Civic Builder.

Humans in the Loop - Mohammed Bin Rashid Space Center, Dubai, UAE

Humans in the Loop is publishing an open access dataset annotated for a joint project with the Mohammed Bin Rashid Space Center in Dubai, UAE. This dataset supports semantic segmentation tasks and can be accessed here.

Boston, MA Labelled Dataset

The Massachusetts Buildings Dataset consists of 151 aerial images of the Boston area, with each image being 1500 × 1500 pixels covering an area of 2.25 square kilometers. This dataset has been utilized in Civic Builder for training and validation purposes. It can be found here.

SpaceNet 2: Building Detection v2

SpaceNet 2 provides high-resolution satellite imagery and labeled building footprint annotations for various regions, including urban areas. Civic Builder has utilized this dataset to enhance its capabilities in building detection and geospatial modeling. More information can be found on the SpaceNet website here.
